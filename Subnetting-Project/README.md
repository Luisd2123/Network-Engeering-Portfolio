# Subnetting and Company IP Design Project

## Overview

This project demonstrates my ability to design and implement a structured IPv4 addressing scheme using VLSM (Variable-Length Subnet Masking). I will take a single network and divide it efficiently into multiple subnets for different departments in the simulated organization using Cisco Packet Tracer. 

## Scenario

A company was assigned the network:

192.168.10.0/24

The company has the following departments:

- Engineering (60 hosts)
- Sales (30 hosts)
- Human Resources (15 hosts)
- IT (10 hosts)
- Accounting (6 hosts)
- Server Room (2 hosts)
